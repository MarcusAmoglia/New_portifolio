# 💻 Marcus Amoglia - Portfolio & Dynamic CMS

Bem-vindo ao repositório do meu portfólio pessoal! Este projeto vai além de uma simples página estática: trata-se de uma aplicação real construída como um ecossistema completo, apresentando minhas habilidades como Desenvolvedor Full Stack / QA Analyst e integrando um **Painel Administrativo (CMS) exclusivo** para gerenciamento de conteúdo em tempo real.

---

## 🚀 O Projeto

A ideia central foi consolidar conhecimentos avançados em **React**, **Vite** e **Tailwind CSS**, transformando o clássico portfólio de desenvolvedor em um produto de software maduro. A aplicação possui dois ecossistemas integrados:
1. **O Portfólio (Client-side):** Uma interface fluida, altamente responsiva, com suporte a animações ricas, alternância de temas e multi-idioma.
2. **O Painel Admin (Dashboard):** Um painel protegido por autenticação que me permite modificar dinamicamente qualquer informação textual, métrica ou decisão de design (como a paleta de cores primárias/secundárias) diretamente pelo navegador, sem necessidade de alterar o código-fonte.

---

## ✨ Funcionalidades Principais

* 🌐 **Troca de Idioma Integrada:** Suporte completo e dinâmico a múltiplos idiomas (Português/Inglês) aplicado a todas as seções.
* 🌓 **Alternância de Temas:** Suporte nativo a Dark Mode e Light Mode com transições suaves.
* 🔐 **Painel Administrativo Protegido:** Tela de login robusta com validações visuais de credenciais para controle seguro dos dados.
* 🎨 **Customização Estética Real-time:** Alteração dinâmica de cores primárias, secundárias e backgrounds diretamente no painel admin, refletindo instantaneamente na interface pública.
* 💾 **Persistência & Reset via JSON:** Sistema de exportação de configurações de estilo para backup e funcionalidade de reset rápido para o padrão de fábrica estruturado em JSON.
* 📱 **Layout Totalmente Responsivo:** Adaptação minuciosa de design para resoluções de desktop, tablet e mobile.
* 🎬 **Animações Fluidas:** Microinterações cuidadosas desenvolvidas com **Framer Motion** para garantir uma experiência de navegação premium sem poluição visual.

---

## 🛠️ Tecnologias Utilizadas

* **Core:** [React](https://react.dev/) + [Vite](https://vitejs.dev/) (Build tool ultra-rápida)
* **Estilização:** [Tailwind CSS](https://tailwindcss.com/) (Utilitários de CSS para design consistente)
* **Animações:** [Framer Motion](https://www.framer.com/motion/) (Animações de componentes estruturais e rotas)
* **Gerenciamento de Estado:** React Context API / Hooks customizados
* **Iconografia:** [Lucide Icons](https://lucide.dev/) / React Icons

## 📦 Como Executar Localmente

Siga os passos abaixo para clonar o repositório e rodar o projeto em sua máquina:

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/MarcusAmoglia/portfolio.git
   ```

2. **Acesse o diretório do projeto:**
   ```bash
   cd portfolio
   ```

3. **Instale as dependências:**
   ```bash
   npm install
   ```

4. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

5. **Abra no navegador:**
   O Vite irá gerar um endereço local (geralmente `http://localhost:5173`). Basta abrir este link no navegador para visualizar a aplicação.
